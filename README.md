# 🎮 NEXUS Gaming Marketplace

NEXUS Gaming es una landing page de vanguardia para una tienda digital de videojuegos. Presenta una estética oscura con acabados neón (Cian y Morado) inspirada en la cultura gamer, combinando un diseño visual impactante con funcionalidades interactivas reales.

## 💎 Funcionalidades Destacadas
* **Sección de Información Corporativa:** Bloque "Sobre Nosotros" que explica los valores y ventajas de comprar en la tienda.
* **Sección de Ofertas:** Espacio destacado para juegos promocionales o *Free-to-Play*.
* **Menús Desplegables Inteligentes:** Uso de la etiqueta nativa `<details>` en HTML para mostrar descripciones, plataformas y requerimientos mínimos de PC sin saturar el diseño.
* **Carrito de Compras Interactivo (Sistema Toggle):** Lógica avanzada en JavaScript utilizando `dataset` para permitir al usuario añadir o remover un juego del carrito con un solo botón, actualizando el contador global dinámicamente y mostrando una confirmación visual (`✓`).

## 🛠️ Tecnologías Utilizadas
* **HTML5** * **CSS Grid & Flexbox:** Estructura de cuadrícula perfecta para las tarjetas de los videojuegos.
* **JavaScript Puro:** Manipulación dinámica de estados y eventos del DOM.

## 📸 Optimización Visual
* Organización de recursos locales en carpetas dedicadas (`logos/`).
* Ajuste de renderizado de imágenes mediante `object-fit: cover` para evitar deformaciones o pixelado en los logos de los videojuegos.
